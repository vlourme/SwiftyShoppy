# SwiftyShoppy
![GitHub](https://img.shields.io/github/license/vlourme/SwiftyShoppy)

> SwiftyShoppy is an API wrapper for shoppy.gg selling service.
> This API is built on top of Moya and aims to reproduce as soon as possible the full API proposed on [shoppy.dev](shoppy.dev).
